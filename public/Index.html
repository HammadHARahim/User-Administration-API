User Administration API (in developement)

This API allows administrators to create, retrieve, update, and delete user records in a MySQL database using Sequelize. It includes robust exception handling for error management.

Base URL

http://localhost:3000/api/v1/users

Endpoints

1. Create User

Method: POST

URL: /users

Request Body: JSON

{
    "username": "string",
    "email": "string",
    "password": "string"
}

Response: JSON

{
    "message": "User created successfully"
}

2. Get User

Method: GET

URL: /users/:id

Response: JSON

{
    "id": "number",
    "username": "string",
    "email": "string"
}

3. Update User

Method: PUT

URL: /users/:id

Request Body: JSON

{
    "username": "string",
    "email": "string"
}

Response: JSON

{
    "message": "User updated successfully"
}

4. Delete User

Method: DELETE

URL: /users/:id

Response: JSON

{
    "message": "User deleted successfully"
}

Error Handling

The API implements exception handling mechanisms to return detailed error messages in JSON format: 

{
    "error": "Error message here"
}

Technologies Used

  • Node.js
  • Sequelize (ORM)
  • MySQL
